## Projeto Final: Órbita Zero 🚀

**Disciplina:** Desenvolvimento para Dispositivos Móveis<br>
**Alunos:** João Victor Nogueira e Sabrina de Oliveira Souza

### 🎯 Objetivo:
- Este projeto tem como objetivo o desenvolvimento de um jogo mobile 2D utilizando o framework Flutter e o motor de jogos 2D Flame, aplicando conceitos de desenvolvimento mobile como controle por gestos, animação contínua e gerenciamento de estado de jogo.

### 🛠️ Tecnologias
- Flutter e Flame.

## 📝 Releases:
### Release 1: Fundação Jogável
- US-01 Controlar a nave com arrasto do dedo
- US-02 Tela rola automaticamente para baixo
- Critério de aceite: O jogador consegue mover a nave com o dedo enquanto o cenário scrolla.

### Release 2: Combate
- US-03 Nave dispara automaticamente
- Critério de aceite: Projéteis são disparados periodicamente sem interação do jogador.

### Release 3: Ciclo Completo de Jogo
- US-04 Indicador de vida da nave
- US-05 Tela de Game Over ao fim da partida
- Critério de aceite: A vida diminui ao tomar dano, e ao zerar exibe a tela de game over.

## ✅ Requisitos Funcionais
- RF-01 O sistema deve permitir mover a nave arrastando o dedo;
- RF-02 A nave não deve ultrapassar os limites da tela;
- RF-03 O cenário deve rolar automaticamente de cima para baixo;
- RF-04 A velocidade do scroll deve ser constante;
- RF-05 O jogador não deve precisar interagir para atirar;
- RF-06 A nave deve ter um indicador visual de vida na tela;
- RF-07 A vida deve diminuir ao colidir com inimigos ou obstáculos;
- RF-08 Ao zerar a vida, o jogo deve exibir a tela de Game Over;
- RF-09 A tela de Game Over deve permitir reiniciar a partida;

## 🔒 Requisitos Não Funcionais
- RNF-01 O app deve rodar em Android (Portabilidade);
- RNF-02 O código deve ser organizado em camadas separadas (UI, lógica, entidades) (Manutenibilidade);
- RNF-03 O jogo deve funcionar completamente offline (Confiabilidade)
